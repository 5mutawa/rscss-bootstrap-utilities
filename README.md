## RCSS Bootstrap utilities

A rewrite for [Bootstrap 4 utilities](https://getbootstrap.com/docs/4.1/)  with RSCSS helpers conventions. 

### Example
```
.mt-0 becomes ._mt-0
```

### To build
```bash
$ npm run compile-css
```
After that, you can copy and use css/bootstrap-utilities.css
